# 🤖 AI Resume Analyzer

An AI-powered web app that analyzes your resume against any job description in seconds.

## ✨ Features
- 🎯 Match Score (0–100)
- ✅ Matched Keywords
- ❌ Missing Keywords  
- 📊 Skills, Experience & Format breakdown
- 💡 5 AI-powered improvement suggestions
